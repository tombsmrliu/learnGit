2.1获取git仓库
   2.1.1获取git仓库
      （1）在现有项目或者目录下导入所有文件到git中
      （2）从一个服务器克隆一个现有的git仓库
   2.1.1.1在现有的目录中初始化仓库
       输入命令：
     $git init
       该命令将创建一个名为.git的子目录，这个子目录包含你的初始化的仓库中所有的必须文件。
       可以通过git add 文件名 来向暂存区添加一个或多个文件，利用git commit来把暂存区的文件列表提交到git仓库，至于怎么查看暂存区的文件俩表以后讨论。
     $git add *.txt
     $git add readme
     $git commit -m "init project version"
